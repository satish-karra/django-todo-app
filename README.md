# Django ToDo Application

## 📌 Project Overview

This project is a web-based ToDo application built using Django. It demonstrates core backend development concepts including CRUD operations, task management, and dynamic data handling using a relational database.

---

## 🚀 Features

* Create, read, update, and delete (CRUD) tasks
* Mark tasks as completed or pending
* Dynamic task updates using Django views
* Clean and structured backend logic
* User-friendly task management interface

---

## 🛠️ Tech Stack

* Python
* Django
* SQLite
* HTML, CSS

---

## 📂 Project Structure

* `todo/` → Core application logic
* `templates/` → HTML templates for UI
* `todo_main/` → Project configuration and settings

---

## ▶️ How to Run

1. Clone the repository
   git clone https://github.com/satish-karra/django-todo-app

2. Navigate to the project
   cd django-todo-app

3. Install dependencies
   pip install -r requirements.txt

4. Run the server
   python manage.py runserver

---

## 📡 Functionalities Implemented

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as done/undone
* Display all tasks dynamically
